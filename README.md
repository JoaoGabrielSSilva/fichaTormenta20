# 🐉 Criador de Fichas - Tormenta 20

Uma aplicação web completa, intuitiva e responsiva para criação e gerenciamento de fichas de personagem para o sistema de RPG **Tormenta 20**. 

Este projeto visa facilitar a vida de mestres e jogadores, permitindo a criação rápida de personagens com exportação direta para PDF (para impressão/leitura) e JSON (para backup ou integração).

![Licença](https://img.shields.io/badge/licença-MIT-green)

## ✨ Funcionalidades

- 📝 **Identidade do Personagem:** Nome, jogador, raça, classe, divindade, alinhamento e nível.
- 📊 **Atributos:** Preenchimento de Força, Agilidade, Intelecto, Presença e Vigor.
- ⚔️ **Dados de Combate:** PV (Pontos de Vida), PD (Pontos de Defesa), CA (Classe de Armadura), deslocamento e resistências.
- 🎯 **Perícias:** Lista completa de perícias do Tormenta 20 com treinamento e bônus.
- 🌟 **Habilidades:** Espaço para poderes de classe, talentos e habilidades raciais.
- 📜 **História:** Campo de texto rico para background, aparência e ideais do personagem.
- 💾 **Exportação Flexível:**
  - 📄 **PDF:** Geração de ficha formatada e pronta para impressão.
  - 📦 **JSON:** Exportação dos dados brutos para backup ou importação futura.


## 📖 Como Usar

1. Acesse a aplicação: https://joaogabrielssilva.github.io/fichaTormenta20/
2. Preencha os dados de identidade do seu personagem.
3. Distribua os valores de atributos conforme a criação de personagem do T20.
4. Preencha as perícias, equipamentos e habilidades.
5. Ao finalizar, clique no botão "Exportar" no final da página ou, se preferir, imprima sua ficha.
